# Real-Time-Data-Acquisition-and-Visualization-System-Embedded-Python-
Real-time data acquisition and visualization from an embedded system using Python.

This project demonstrates serial data communication between an embedded system and a Python environment. The acquired data is processed and visualized in real time.

The system includes:
- Continuous data streaming via serial communication  
- Real-time visualization using Python  

Example implementations are provided in the repository.

#HARDWARE: 
- ESP32-32D microcontroller kit. (Any kind of microcontroller with serial communication feature should work.)
- DHT22 (Temperature-Humidity monitoring)
- Potentiometer. (Raw ADC monitoring)
- USB A to Micro-USB cable.
- Basic components. (Breadboard, jumpers, resistors and capacitors.)


#SOFTWARE:
- Arduino IDE
- Python (Matplotlib, PySerial included.)

Matplotlib : https://matplotlib.org/ 
PySerial : https://pypi.org/project/pyserial/
